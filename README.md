# AI Search and Optimization Algorithms in Python

## Overview

This repository contains implementations of classical Artificial Intelligence algorithms covering search strategies, constraint satisfaction problems (CSP), and genetic algorithms.

Each algorithm is implemented in Python to demonstrate its working, logic, and problem-solving approach.

---

## Why I Built This

I built this repository to:

* Understand core AI problem-solving techniques
* Implement search algorithms from scratch
* Learn optimization and evolutionary computation methods
* Gain hands-on experience with CSP and graph traversal problems

---

## Project Structure

```plaintext id="ai_structure"
AI-Repo/
│── 4Queen.py                  # N-Queens problem using backtracking/CSP
│── Astar.py                   # A* search algorithm
│── bfs.py                     # Breadth First Search
│── dfs.py                     # Depth First Search
│── ucs.py                     # Uniform Cost Search
│── csp_map.py                # Constraint Satisfaction Problem (map coloring)

│── GAstring.py               # Genetic Algorithm (string evolution)
│── genetic algorithm.py      # Additional GA implementation
```

## Algorithms Included

### Search Algorithms

* Breadth First Search (BFS)
* Depth First Search (DFS)
* Uniform Cost Search (UCS)
* A* Search Algorithm

---

### Constraint Satisfaction Problems (CSP)

* N-Queens Problem (4-Queen implementation)
* Map Coloring Problem using CSP techniques

---

### Genetic Algorithms

* Function optimization using GA
* String evolution using GA
* Selection, crossover, and mutation operations

